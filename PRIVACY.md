# Privacy Policy

## Overview

This extension provides pipeline tasks for Azure DevOps.
All functionality runs within Azure DevOps infrastructure.

For more information on security and privacy of Azure DevOps refer to https://azure.microsoft.com/en-us/support/legal.

## Data Storage

All possible data written by the extension is stored within Azure DevOps infrastructure.
No data is transferred outside of Azure DevOps.

## Privacy

The extension does not collect, store, or transmit any personally identifiable information.
No data accessible to the extension within Azure DevOps will be transferred to external services.
